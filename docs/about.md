欢迎来到我的个人博客。这里是我的**个人知识库**和**技术博客**，用于记录学习、分享思考、收藏有用资源。

## 网站内容

- **Notes** - 个人学习笔记，涵盖前端、后端、运维等
- **Tools** - 实用网站导航，AI 工具、开发工具、学习资源
- **Blog** - 技术博客与生活随笔

## 技术栈

本网站使用以下技术搭建：

- [VitePress](https://vitepress.dev/) - 基于 Vite 和 Vue 的静态站点生成器
- [vitepress-theme-open17](https://github.com/open17/vitepress-theme-open17) - 简洁美观的博客主题
- [Markdown](https://www.markdownguide.org/) - 内容编写
