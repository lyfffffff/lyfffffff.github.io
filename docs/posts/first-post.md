# 欢迎

这是使用 **VitePress** 和 **vitepress-theme-open17** 主题搭建的个人博客。

## 博客功能

- 📅 **日期**：每篇文章支持发布日期
- 🏷️ **标签**：方便分类和检索
- ⏱️ **阅读时间**：自动估算阅读时长
- 🌙 **暗色模式**：支持亮色/暗色主题切换

## 接下来

我会在这里分享：

1. **个人笔记** - 前端、后端、运维等学习记录
2. **技术博客** - 项目实践与问题解决
3. **工具导航** - 常用网站与资源整理

感谢访问！
